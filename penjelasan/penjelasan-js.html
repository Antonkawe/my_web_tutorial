Penjelasan JavaScript

1. Var

Memiliki cakupan global atau fungsi, dan dapat diperbarui dan dideklarasikan ulang. Var diinisialisasi dengan undefined.

2. Let

Memiliki cakupan blok, dan dapat diperbarui tetapi tidak dideklarasikan ulang. Let tidak diinisialisasi.

3. Const

Memiliki cakupan blok, dan tidak dapat diperbarui maupun dideklarasikan ulang. Const harus diinisialisasi selama deklarasi.

4. Tipe Data Primitive

Termasuk string, number, boolean, null, undefined, dan symbol. Nilai-nilai ini tidak dapat diubah setelah mereka diciptakan.

5. Tipe Data Non-Primitive

Seperti objek, array, dan function. Tipe data ini dapat diubah dan memiliki sifat yang lebih kompleks.

6. String

String adalah urutan karakter, dikelilingi oleh tanda kutip tunggal, ganda, atau backtick (untuk template literal).

7. Number

Number mencakup bilangan bulat dan pecahan, baik dalam bentuk desimal maupun eksponensial.

8. Boolean

Boolean hanya memiliki dua nilai: true atau false.

9. Null

Null adalah nilai kosong atau tidak ada yang secara eksplisit diatur oleh programmer.

10. Undefined

Undefined menunjukkan bahwa suatu variabel belum diberikan nilai.

11. Symbol

Symbol adalah tipe data primitif yang digunakan untuk membuat identifier yang unik dan tidak dapat diubah.

12. Object

Object adalah koleksi pasangan key-value yang dapat menyimpan berbagai tipe data.

13. Array

Array adalah tipe data yang digunakan untuk menyimpan daftar nilai dalam urutan tertentu.

14. Function

Function adalah blok kode yang dirancang untuk melakukan tugas tertentu, dan dapat dipanggil dengan nama tertentu.

15. If Statement

If statement digunakan untuk menjalankan blok kode hanya jika kondisi tertentu terpenuhi.

16. Else Statement

Else digunakan bersama if untuk menjalankan blok kode ketika kondisi if tidak terpenuhi.

17. Else If

Else if digunakan untuk menguji beberapa kondisi, dan akan dijalankan jika kondisi sebelumnya gagal.

18. Switch Statement

Switch digunakan untuk memilih salah satu dari beberapa blok kode yang akan dieksekusi berdasarkan nilai tertentu.

19. For Loop

For loop digunakan untuk menjalankan blok kode yang sama beberapa kali dengan kondisi tertentu.

20. While Loop

While loop menjalankan blok kode selama kondisi tertentu terpenuhi.

21. Do-While Loop

Do-while loop mirip dengan while loop, namun menjalankan blok kode setidaknya sekali sebelum memeriksa kondisi.

22. Break

Break digunakan untuk keluar dari sebuah loop atau switch statement lebih awal.

23. Continue

Continue digunakan untuk melanjutkan iterasi berikutnya dalam sebuah loop, melewati sisa blok kode di bawahnya.

24. Return

Return digunakan untuk mengembalikan nilai dari fungsi.

25. Arrow Function

Arrow function adalah cara singkat untuk mendeklarasikan fungsi di JavaScript.

26. Callback Function

Callback function adalah fungsi yang dipanggil sebagai parameter dari fungsi lain.

27. Event Listener

Event listener digunakan untuk mendengarkan dan merespon peristiwa (events) yang terjadi pada elemen HTML.

28. DOM (Document Object Model)

DOM adalah representasi struktur dokumen HTML atau XML yang memungkinkan JavaScript berinteraksi dengan elemen-elemen di dalamnya.

29. Local Storage

Local storage adalah fitur di browser yang memungkinkan penyimpanan data secara lokal di sisi klien.

30. Session Storage

Session storage digunakan untuk menyimpan data sementara selama sesi browser berlangsung.

31. JSON (JavaScript Object Notation)

JSON adalah format data yang digunakan untuk menyimpan dan mentransfer data antar aplikasi atau sistem.

32. Promise

Promise adalah objek yang mewakili hasil dari operasi asinkron yang belum selesai, tapi diharapkan akan selesai di masa depan.

33. Async/Await

Async/Await adalah cara untuk menangani operasi asinkron dengan lebih mudah, menggunakan syntax seperti fungsi synchronous.

34. Fetch API

Fetch API digunakan untuk membuat permintaan HTTP dan menangani respons dalam JavaScript secara asinkron.

35. Class

Class adalah template untuk membuat objek dengan metode dan properti yang sama.

36. Constructor

Constructor adalah metode khusus dalam class yang digunakan untuk menginisialisasi objek yang baru dibuat.

37. Inheritance

Inheritance adalah konsep di mana class baru mewarisi properti dan metode dari class lain yang lebih umum.

38. Super

Super digunakan untuk memanggil metode dari class induk di dalam subclass.

39. Module

Module adalah bagian kode yang dapat diekspor dan diimpor untuk digunakan kembali di tempat lain dalam aplikasi JavaScript.

40. Export

Export digunakan untuk membagikan variabel, fungsi, atau objek dari satu file ke file lain dalam JavaScript.

41. Import

Import digunakan untuk membawa modul, fungsi, atau variabel yang telah diekspor ke dalam file yang lain.

42. Template Literals

Template literals adalah fitur di JavaScript untuk menyisipkan ekspresi dalam string dengan menggunakan tanda backtick (`).

43. Destructuring

Destructuring adalah cara untuk mengekstrak nilai dari array atau objek dan menyimpannya ke dalam variabel terpisah.

44. Spread Operator

Spread operator digunakan untuk menyalin nilai dari array atau objek ke dalam array atau objek lain.

45. Rest Parameter

Rest parameter memungkinkan kita untuk mengumpulkan argumen yang tidak terdefinisi sebelumnya menjadi sebuah array.

46. Set

Set adalah koleksi nilai unik, yang artinya tidak ada nilai yang terduplikasi di dalamnya.

47. Map

Map adalah koleksi pasangan key-value, di mana key dapat berupa tipe data apa pun dan nilainya dapat diubah.

48. WeakSet

WeakSet adalah koleksi nilai unik yang tidak menghalangi garbage collection dan hanya dapat berisi objek.

49. WeakMap

WeakMap adalah koleksi pasangan key-value yang hanya dapat menggunakan objek sebagai key dan tidak mencegah objek key untuk dihapus oleh garbage collection.

50. SetTimeout

SetTimeout digunakan untuk menunda eksekusi suatu fungsi setelah waktu yang ditentukan dalam milidetik.

51. SetInterval

SetInterval digunakan untuk mengeksekusi fungsi berulang-ulang dalam interval waktu tertentu (milidetik).

52. ClearTimeout

ClearTimeout digunakan untuk menghentikan eksekusi fungsi yang dijadwalkan dengan setTimeout.

53. ClearInterval

ClearInterval digunakan untuk menghentikan eksekusi berulang dari fungsi yang dijadwalkan dengan setInterval.

54. Error Handling

Error handling adalah teknik untuk menangkap kesalahan dengan blok try-catch, dan melakukan tindakan saat kesalahan terjadi.

55. Try-Catch

Try-catch adalah blok kode yang menangani kesalahan (error) tanpa menghentikan eksekusi seluruh program.

56. Finally

Finally adalah bagian dari blok try-catch yang akan selalu dieksekusi, terlepas dari apakah ada kesalahan atau tidak.

57. Throw

Throw digunakan untuk membangkitkan atau melemparkan error yang dapat ditangkap oleh blok try-catch.

58. Asynchronous JavaScript

Asynchronous JavaScript mengacu pada metode pemrograman di mana kode tidak dieksekusi secara berurutan.

59. Event Loop

Event loop adalah mekanisme yang menangani eksekusi kode, pengumpulan event, dan subtugas asinkron dalam JavaScript.

60. Callback Hell

Callback hell adalah situasi di mana callback bertumpuk terlalu banyak, sehingga sulit dibaca dan dikelola.

61. Hoisting

Hoisting adalah mekanisme di mana deklarasi variabel dan fungsi dipindahkan ke bagian atas cakupan saat eksekusi.

62. Scope

Scope adalah konsep yang menentukan aksesibilitas variabel di berbagai bagian kode.

63. Global Scope

Global scope adalah cakupan variabel yang dapat diakses di mana saja dalam program JavaScript.

64. Local Scope

Local scope adalah cakupan variabel yang hanya dapat diakses dalam fungsi atau blok tertentu.

65. Lexical Scope

Lexical scope mengacu pada bagaimana JavaScript mencari variabel dalam konteks yang lebih tinggi saat mereka tidak ditemukan dalam scope saat ini.

66. Closure

Closure adalah fitur yang memungkinkan fungsi untuk mengakses variabel di luar cakupannya setelah fungsi selesai dijalankan.

67. Pure Function

Pure function adalah fungsi yang memberikan hasil yang sama setiap kali dipanggil dengan argumen yang sama, tanpa efek samping.

68. IIFE (Immediately Invoked Function Expression)

IIFE adalah fungsi yang dieksekusi segera setelah dideklarasikan, tanpa perlu dipanggil lagi.

69. Memoization

Memoization adalah teknik pengoptimalan yang menyimpan hasil dari fungsi untuk menghindari komputasi berulang.

70. Recursion

Recursion adalah teknik pemanggilan fungsi dalam dirinya sendiri untuk memecahkan masalah yang dapat dibagi menjadi sub-masalah.

71. Higher-Order Function

Higher-order function adalah fungsi yang dapat menerima fungsi lain sebagai argumen atau mengembalikan fungsi sebagai hasil.

72. Map

Map adalah metode array yang digunakan untuk membuat array baru dengan hasil dari pemanggilan fungsi pada setiap elemen.

73. Filter

Filter adalah metode array yang digunakan untuk membuat array baru yang hanya berisi elemen yang memenuhi kondisi tertentu.

74. Reduce

Reduce adalah metode array yang mengakumulasi nilai dari elemen array menjadi satu nilai akhir berdasarkan fungsi yang diberikan.

75. This

This mengacu pada konteks eksekusi saat ini dalam fungsi atau objek.

76. Arrow Function (Lexical 'this')

Arrow function menangkap nilai 'this' dari lingkup di mana ia dideklarasikan, bukan dari lingkup di mana ia dipanggil.

77. Strict Mode

Strict mode adalah mode yang lebih ketat di JavaScript yang membantu menemukan kesalahan dan mencegah beberapa perilaku berisiko.

78. Proxy

Proxy adalah objek yang memungkinkan kita untuk mengatur tindakan khusus untuk operasi dasar (seperti membaca atau menulis properti).

79. Reflect

Reflect adalah objek yang menyediakan metode untuk mengganggu dan menyesuaikan operasi objek di JavaScript.

80. Generator Function

Generator function adalah fungsi yang dapat berhenti sementara dan melanjutkan eksekusinya, menggunakan kata kunci yield.

81. Iterator

Iterator adalah objek yang menyediakan metode untuk mengakses elemen dalam urutan tertentu, satu per satu.

82. Symbol.iterator

Symbol.iterator adalah properti yang memungkinkan sebuah objek digunakan dalam loop for...of, seperti pada array.

83. TypedArray

TypedArray adalah array yang efisien untuk memanipulasi data biner mentah, berguna untuk operasi di tingkat lebih rendah.

84. Proxy Trap

Proxy trap adalah metode yang digunakan untuk mengganggu dan menyesuaikan operasi objek tertentu dalam proxy.

85. Weak Reference

Weak Reference adalah referensi ke objek yang tidak mencegah objek tersebut dihapus dari memori jika tidak digunakan lagi.

86. WeakMap

WeakMap adalah koleksi pasangan kunci-nilai di mana kunci hanya bisa berupa objek, dan nilai tidak akan mencegah objek dari penghapusan otomatis oleh garbage collection.

87. WeakSet

WeakSet adalah kumpulan unik dari objek yang hanya menyimpan referensi lemah, sehingga memungkinkan objek dihapus dari memori jika tidak lagi diperlukan.

88. Async/Await

Async/await adalah cara menulis kode asinkron yang lebih bersih dan mudah dibaca dibandingkan menggunakan promise chaining.

89. Module

Module adalah fitur JavaScript yang memungkinkan kita membagi kode ke dalam file-file terpisah dan mengimpor hanya yang diperlukan.

90. Import

Import digunakan untuk mengambil modul atau bagian dari modul ke dalam file JavaScript lain.

91. Export

Export digunakan untuk mengekspor variabel, fungsi, atau kelas dari suatu modul agar bisa diimpor oleh modul lain.

92. Tree Shaking

Tree shaking adalah teknik untuk menghapus kode yang tidak digunakan dari bundel akhir JavaScript guna mengoptimalkan ukuran file.

93. Dynamic Import

Dynamic import memungkinkan pengimporan modul secara dinamis pada waktu runtime, yang berguna untuk menghemat memori.

94. WebSocket

WebSocket adalah protokol komunikasi yang memungkinkan koneksi real-time antara klien dan server.

95. Service Worker

Service Worker adalah skrip yang berjalan di latar belakang browser dan memungkinkan caching serta fitur offline.

96. IndexedDB

IndexedDB adalah database lokal pada browser yang memungkinkan penyimpanan data dalam jumlah besar untuk aplikasi web.

97. Shadow DOM

Shadow DOM adalah bagian dari Web Components yang memungkinkan pembuatan DOM terisolasi untuk komponen.

98. Custom Elements

Custom Elements adalah bagian dari Web Components yang memungkinkan pembuatan elemen HTML khusus dengan perilaku tertentu.

99. Template

Template adalah elemen HTML yang berisi struktur DOM yang tidak langsung dirender ke halaman sampai dipanggil melalui JavaScript.

100. Event Delegation

Event delegation adalah teknik menangani event pada elemen induk, yang secara otomatis mencakup elemen anaknya.
