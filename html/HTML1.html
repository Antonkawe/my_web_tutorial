1. Struktur Dasar HTML


<!-- Struktur dasar dokumen HTML -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Dokumen HTML</title>
</head>
<body>
    <h1>Selamat datang di HTML!</h1>
    <p>Ini adalah paragraf pertama dalam HTML.</p>
</body>
</html>

Penjelasan: Ini adalah struktur dasar dari dokumen HTML.

2. Penggunaan Tag Paragraf & Heading


<!-- Menggunakan tag paragraf dan heading -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Paragraf dan Heading</title>
</head>
<body>
    <h1>Judul Utama</h1>
    <h2>Sub Judul</h2>
    <p>Ini adalah paragraf pertama.</p>
</body>
</html>

Penjelasan: `h1` digunakan untuk judul utama, dan `h2` untuk subjudul.

3. Tag Gambar HTML


<!-- Menambahkan gambar dengan tag img -->
<img src="path/to/image.jpg" alt="Deskripsi gambar">

Penjelasan: Tag `img` digunakan untuk menambahkan gambar.

4. Membuat Tautan dengan Tag Anchor


<!-- Menambahkan tautan dengan tag a -->
<a href="https://www.example.com">Kunjungi Situs Web</a>

Penjelasan: Tag `a` digunakan untuk membuat tautan.

5. Membuat Daftar Tak Berurut


<!-- Membuat daftar tidak berurutan -->
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

Penjelasan: `ul` digunakan untuk daftar tidak berurutan, dan `li` untuk item daftar.

6. Membuat Daftar Berurutan


<!-- Membuat daftar berurutan -->
<ol>
    <li>Item 1</li>
    <li>Item 2</li>
</ol>

Penjelasan: `ol` digunakan untuk daftar berurutan.

7. Membuat Tabel


<!-- Membuat tabel sederhana -->
<table>
    <tr>
        <th>Nama</th>
        <th>Usia</th>
    </tr>
    <tr>
        <td>Anton</td>
        <td>16</td>
    </tr>
</table>

Penjelasan: Tag `table` digunakan untuk membuat tabel.

8. Formulir Dasar


<!-- Membuat form dasar -->
<form>
    <label for="name">Nama:</label>
    <input type="text" id="name" name="name">
    <button type="submit">Kirim</button>
</form>

Penjelasan: Tag `form` digunakan untuk membuat formulir input data.

9. Memasukkan Video ke HTML


<!-- Menambahkan video -->
<video controls>
    <source src="path/to/video.mp4" type="video/mp4">
</video>

Penjelasan: Tag `video` digunakan untuk menampilkan video dengan kontrol play/pause.

10. Mengatur Teks dengan Tag CSS Inline


<!-- Mengatur teks dengan CSS inline -->
<p style="color: blue; font-size: 18px;">Teks berwarna biru dan ukuran font 18px</p>

Penjelasan: CSS inline memungkinkan Anda untuk memberi gaya pada elemen secara langsung di dalam tag HTML.
