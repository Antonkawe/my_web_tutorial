1. Penggunaan Tag Semantic HTML


<!-- Menggunakan tag semantic untuk struktur yang lebih baik -->
<header><h1>Judul Artikel</h1></header>
<nav><a href="#content">Lompat ke konten</a></nav>
<section id="content">
    <article><p>Konten utama artikel di sini.</p></article>
</section>

Penjelasan: Tag seperti `header`, `nav`, dan `article` membantu SEO dan aksesibilitas.

2. Menggunakan CSS Flexbox untuk Layout


<!-- Membuat layout responsif dengan Flexbox -->
<div style="display: flex; gap: 10px;">
    <div>Kolom 1</div>
    <div>Kolom 2</div>
    <div>Kolom 3</div>
</div>

Penjelasan: Flexbox memudahkan pengaturan layout fleksibel untuk elemen-elemen.

3. Menggunakan CSS Grid


<!-- Layout dengan CSS Grid -->
<div style="display: grid; grid-template-columns: repeat(3, 1fr); gap: 10px;">
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
</div>

Penjelasan: CSS Grid memungkinkan layout 2D yang lebih kompleks.

4. Membuat Animasi dengan CSS


<!-- Menambahkan animasi dengan CSS -->
<style>
    @keyframes fadeIn {
        from { opacity: 0; }
        to { opacity: 1; }
    }
    .fade-in { animation: fadeIn 2s ease-in; }
</style>
<p class="fade-in">Teks muncul perlahan</p>

Penjelasan: Menggunakan `@keyframes` untuk membuat animasi CSS sederhana.

5. Menggunakan Tag <iframe>


<!-- Menyisipkan konten lain dengan iframe -->
<iframe src="https://www.example.com" width="600" height="400"></iframe>

Penjelasan: Tag `iframe` digunakan untuk menyisipkan halaman atau konten lain.

6. Menambahkan Video dari YouTube dengan Embed


<!-- Menambahkan video YouTube -->
<iframe width="560" height="315" src="https://www.youtube.com/embed/video_id" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

Penjelasan: Menggunakan `iframe` untuk embed video dari platform lain.

7. Membuat Formulir yang Lebih Kompleks


<!-- Formulir dengan berbagai tipe input -->
<form>
    <label>Nama: <input type="text" name="name"></label>
    <label>Email: <input type="email" name="email"></label>
    <label>Password: <input type="password" name="password"></label>
    <button type="submit">Kirim</button>
</form>

Penjelasan: Menggunakan berbagai tipe input untuk kebutuhan form yang lebih kompleks.

8. Membuat Modal dengan JavaScript


<!-- Membuat modal popup dengan JavaScript -->
<button onclick="document.getElementById('modal').style.display='block'">Buka Modal</button>
<div id="modal" style="display:none;">
    <p>Ini adalah modal popup</p>
    <button onclick="document.getElementById('modal').style.display='none'">Tutup</button>
</div>

Penjelasan: Membuat pop-up sederhana dengan JavaScript.

9. Menggunakan Local Storage di JavaScript


<!-- Menyimpan data ke Local Storage -->
<script>
    localStorage.setItem('name', 'Anton');
    document.write(localStorage.getItem('name'));
</script>

Penjelasan: Local Storage memungkinkan penyimpanan data pada browser.

10. Dasar-Dasar Fetch API untuk Mengambil Data dari API


<!-- Mengambil data dari API dengan Fetch -->
<script>
    fetch('https://jsonplaceholder.typicode.com/posts/1')
    .then(response => response.json())
    .then(data => console.log(data));
</script>

Penjelasan: Fetch API digunakan untuk mengambil data dari server atau API.
