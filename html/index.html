HTML - 1: Basic Structure


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Webpage</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
</body>
</html>

This tutorial covers the basic HTML structure, including the use of `DOCTYPE`, `html`, `head`, and `body` tags.

HTML - 2: Forms and Inputs


<form>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="Submit">
</form>

Learn how to create basic forms and input fields in HTML, allowing users to submit data through your website.

HTML - 3: Lists and Links


<ul>
    <li>Home</li>
    <li>About</li>
    <li>Contact</li>
</ul>

<a href="https://example.com">Visit Example</a>

This tutorial teaches you how to create lists and add hyperlinks to your HTML pages, which is essential for navigation.

HTML - 4: Tables and Images


<table>

        <th>Name</th>

    </tr>
    <tr>
        <td>Anton</td>
        <td>16</td>

This section covers how to create tables and insert images into your HTML documents.
