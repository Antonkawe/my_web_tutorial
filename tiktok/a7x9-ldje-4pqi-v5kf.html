AlphaCoder API Documentation

Dokumentasi API untuk Unduh Video TikTok Tanpa Watermark

API Endpoint
Untuk mengunduh video TikTok tanpa watermark, gunakan endpoint berikut: 
GET https://www.antoncodder.online/api/tiktok?apikey={API_KEYS}&url={TIKTOK_URL}
Request Parameters
Berikut adalah parameter yang dapat digunakan dalam permintaan API: 
url (required): URL video TikTok yang ingin diunduh.
API Key

Response
API ini akan mengembalikan respons JSON dengan informasi video: 

{
  "status": "success",
  "project": "AlphaCoder",
  "owner": "Anton",
  "video": {
    "id": "ID_VIDEOS",
    "region": "REGION_VIDEOS",
    "title": "TITLE_VIDEOS",
    "url": "https://example.com/video",
    "duration": DURATION_VIDEOS
  }
}
Contoh Penggunaan
Berikut adalah contoh cara menggunakan API ini dengan JavaScript menggunakan fetch: 

fetch('https://www.antoncodder.online/api/tiktok?apikey=AlphaCoder03&url=https://www.tiktok.com/example')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error:', error));
Kebijakan Penggunaan
Harap perhatikan kebijakan penggunaan berikut saat menggunakan API ini:
  • Gunakan API ini hanya untuk tujuan legal dan sesuai dengan hukum yang berlaku.
  • Jangan membagikan API Key Anda kepada pihak lain untuk mencegah penyalahgunaan.
  • Batasi penggunaan sesuai dengan batas rate-limit API.
  • AlphaCoder tidak bertanggung jawab atas penyalahgunaan layanan ini oleh pihak ketiga.
