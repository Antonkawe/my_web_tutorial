1. Hello World


// Menampilkan teks "Hello, World!" ke console
console.log("Hello, World!");

Penjelasan: Kode ini mencetak teks "Hello, World!" ke console, yang merupakan cara pertama yang umum untuk memulai belajar pemrograman di banyak bahasa.

2. Variabel dan Tipe Data


// Mendeklarasikan variabel dengan tipe data string dan number
let name = "Anton";
let age = 16;
console.log(name, age);

Penjelasan: Di sini kita mendeklarasikan dua variabel menggunakan `let`. Variabel `name` menyimpan nilai string dan `age` menyimpan angka. Kode ini kemudian mencetak keduanya ke console.

3. Fungsi Dasar


// Fungsi greet yang menerima parameter name
function greet(name) {
    console.log("Hello, " + name + "!");
}
greet("Anton");

Penjelasan: Fungsi `greet` didefinisikan untuk menerima parameter `name` dan mencetak pesan sapaan menggunakan `console.log()`.

4. Pengkondisian (If Statement)


// Pengkondisian untuk mengecek apakah number lebih besar dari 5
let number = 10;
if (number > 5) {
    console.log("Number is greater than 5");
} else {
    console.log("Number is 5 or less");
}

Penjelasan: Di sini, kode memeriksa apakah variabel `number` lebih besar dari 5 dan mencetak hasilnya berdasarkan kondisi tersebut.

5. Looping (For Loop)


// Loop untuk mencetak angka dari 1 sampai 5
for (let i = 1; i <= 5; i++) {
    console.log(i);
}

Penjelasan: `for` loop digunakan untuk menjalankan kode secara berulang. Dalam hal ini, kode ini akan mencetak angka dari 1 hingga 5.

6. Array dan Looping Array


// Array dengan beberapa nilai dan loop untuk mencetaknya
let fruits = ["Apple", "Banana", "Cherry"];
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

Penjelasan: Array digunakan untuk menyimpan beberapa elemen. Di sini kita menggunakan loop untuk mencetak setiap elemen dari array `fruits`.

7. Objek dalam JavaScript


// Objek dengan properti dan metode
let person = {
    name: "Anton",
    age: 16,
    greet: function() {
        console.log("Hello, " + this.name);
    }
};
person.greet();

Penjelasan: Objek dalam JavaScript memungkinkan kita untuk menyimpan berbagai properti dan metode. Di sini, objek `person` memiliki properti `name` dan `age`, serta metode `greet` yang digunakan untuk menyapa.

8. Arrow Function


// Arrow function untuk penjumlahan dua angka
const add = (a, b) => a + b;
console.log(add(5, 3));

Penjelasan: Arrow function adalah cara singkat untuk menulis fungsi di JavaScript. Contoh ini menunjukkan cara menggunakan arrow function untuk menjumlahkan dua angka.

9. Promise dalam JavaScript


// Membuat promise untuk menangani operasi asinkron
let promise = new Promise((resolve, reject) => {
    let success = true;
    if (success) {
        resolve("Operation was successful!");
    } else {
        reject("Operation failed");
    }
});
promise.then(result => console.log(result)).catch(error => console.log(error));

Penjelasan: Promise digunakan untuk menangani operasi asinkron. Di sini kita membuat promise yang berhasil jika kondisi `success` bernilai true, dan gagal jika sebaliknya.

10. Event Handling


// Menambahkan event listener untuk tombol
document.getElementById("myButton").addEventListener("click", function() {
    alert("Button clicked!");
});

Penjelasan: Kode ini menambahkan event listener pada tombol dengan id `myButton`. Ketika tombol diklik, sebuah alert akan muncul.
