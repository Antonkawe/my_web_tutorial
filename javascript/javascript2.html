1. Asynchronous JavaScript (setTimeout)


// Menunda eksekusi kode dengan setTimeout
console.log("Start");
setTimeout(() => {
    console.log("Delayed message");
}, 2000); // 2000ms = 2 detik
console.log("End");

Penjelasan: `setTimeout` digunakan untuk menunda eksekusi kode dalam waktu tertentu. Dalam contoh ini, "Delayed message" akan muncul setelah 2 detik, sementara "Start" dan "End" dicetak lebih dulu.

2. Promise dengan Fetch API


// Mengambil data dari API menggunakan Fetch
fetch('https://jsonplaceholder.typicode.com/posts')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.log("Error:", error));

Penjelasan: Di sini kita menggunakan `fetch` untuk mengambil data dari API dan mengonversinya ke dalam format JSON. Kita juga menangani kemungkinan error dengan `catch`.

3. Async/Await


// Menggunakan async/await untuk menangani asinkron
async function getPosts() {
    try {
        let response = await fetch('https://jsonplaceholder.typicode.com/posts');
        let data = await response.json();
        console.log(data);
    } catch (error) {
        console.log("Error:", error);
    }
}
getPosts();

Penjelasan: Dengan `async/await`, kita dapat menulis kode asinkron dengan cara yang lebih bersih dan lebih mudah dipahami, tanpa perlu menggunakan `.then()`.

4. Closure dalam JavaScript


// Closure adalah fungsi yang memiliki akses ke variabel di luar fungsi
function outerFunction(outerVariable) {
    return function innerFunction(innerVariable) {
        console.log(outerVariable, innerVariable);
    };
}

let myClosure = outerFunction("Hello");
myClosure("World");

Penjelasan: Closure terjadi ketika sebuah fungsi memiliki akses ke variabel yang dideklarasikan di luar fungsi tersebut. Dalam contoh ini, `innerFunction` memiliki akses ke `outerVariable` yang didefinisikan dalam `outerFunction`.

5. Debouncing Function


// Menunda eksekusi fungsi hingga setelah selesai mengetik
let debounceTimer;
function debounce(func, delay) {
    clearTimeout(debounceTimer);
    debounceTimer = setTimeout(func, delay);
}

function sayHello() {
    console.log("Hello after typing...");
}

document.getElementById("inputField").addEventListener('keyup', () => {
    debounce(sayHello, 500);
});

Penjelasan: Debouncing digunakan untuk menunda eksekusi suatu fungsi hingga pengguna selesai melakukan aksi tertentu, seperti mengetik. Dalam contoh ini, fungsi `sayHello` hanya akan dipanggil setelah pengguna berhenti mengetik selama 500ms.

6. Map, Filter, dan Reduce


// Menggunakan map, filter, dan reduce untuk memproses array
let numbers = [1, 2, 3, 4, 5];

let squared = numbers.map(x => x * x);
console.log(squared);

let evenNumbers = numbers.filter(x => x % 2 === 0);
console.log(evenNumbers);

let sum = numbers.reduce((acc, x) => acc + x, 0);
console.log(sum);

Penjelasan: `map` digunakan untuk mengubah setiap elemen dalam array, `filter` untuk menyaring elemen, dan `reduce` untuk menggabungkan elemen-elemen menjadi satu nilai. Ketiganya adalah metode penting untuk memanipulasi array di JavaScript.

7. Destructuring Assignment


// Menggunakan destructuring untuk ekstrak nilai dari objek
let person = { name: "Anton", age: 16 };
let { name, age } = person;
console.log(name, age);

Penjelasan: Destructuring adalah cara yang lebih singkat untuk mengekstrak nilai dari objek atau array. Di sini, kita mengekstrak `name` dan `age` dari objek `person`.

8. Arrow Functions with This


// Perbedaan antara fungsi biasa dan arrow function dengan "this"
function regularFunction() {
    console.log(this);
}

const arrowFunction = () => {
    console.log(this);
};

regularFunction(); // this menunjuk pada global object
arrowFunction(); // this menunjuk pada konteks di mana arrow function didefinisikan

Penjelasan: Fungsi biasa menggunakan nilai `this` dari objek tempat fungsi tersebut dipanggil, sementara `arrow function` tidak memiliki nilai `this` sendiri, dan akan mewarisi nilai `this` dari konteks di mana fungsi tersebut didefinisikan.

9. IIFE (Immediately Invoked Function Expression)


// IIFE adalah fungsi yang didefinisikan dan langsung dipanggil
(function() {
    console.log("This is an IIFE");
})();

Penjelasan: IIFE adalah pola di mana fungsi didefinisikan dan langsung dipanggil setelah dideklarasikan. Ini sering digunakan untuk membuat fungsi yang tidak mempengaruhi ruang lingkup global.

10. Modules in JavaScript


// Menggunakan import dan export untuk modul JavaScript
// File: math.js
export function add(a, b) {
    return a + b;
}

// File utama
import { add } from './math.js';
console.log(add(2, 3));

Penjelasan: Modul di JavaScript memungkinkan kita untuk membagi kode menjadi bagian-bagian terpisah yang dapat digunakan di berbagai file. Di sini, kita mengimpor fungsi `add` dari file `math.js` dan menggunakannya dalam file utama.
