JavaScript - Functions & Loops


const printNumbers = (max) => {
    for (let i = 1; i <= max; i++) {
        console.log(`Nomor: ${i}`);
    }
};
printNumbers(5);

Pelajari bagaimana fungsi dan loop bekerja untuk memproses data berulang kali.

JavaScript - Objects & Arrays


const person = {
    name: 'Anton',
    age: 16,
    skills: ['JavaScript', 'Node.js', 'CSS']
};
person.skills.forEach(skill => console.log(`Skill: ${skill}`));

Gunakan objek dan array untuk mengatur data dengan lebih efektif.

JavaScript - Advanced Concepts


const fetchData = async () => {
    try {
        const response = await fetch('https://jsonplaceholder.typicode.com/posts');
        const data = await response.json();
        console.log('Data fetched:', data.slice(0, 5));
    } catch (error) {
        console.error('Error fetching data:', error);
    }
};
fetchData();

Pelajari konsep lanjutan seperti asynchronous programming dan error handling untuk aplikasi yang lebih kompleks.
