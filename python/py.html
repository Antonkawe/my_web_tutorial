Pengenalan Python

Python adalah bahasa pemrograman yang mudah dipelajari dan sangat kuat. Python sering digunakan dalam berbagai bidang, seperti pengembangan web, analisis data, kecerdasan buatan, dan lainnya.

Instalasi Python

Untuk mulai menggunakan Python, kamu harus menginstalnya terlebih dahulu. Ikuti langkah-langkah berikut:

    1. Kunjungi halaman resmi Python di python.org.

    2. Pilih versi terbaru untuk sistem operasi kamu.

    3. Ikuti instruksi instalasi yang muncul di layar.

Sintaks Dasar Python

Python memiliki sintaks yang sederhana dan mudah dibaca. Berikut adalah contoh penggunaan variabel dan operator:


# Variabel dan Tipe Data
x = 5   # Integer
y = 3.2 # Float
nama = "Anton"  # String

# Operator
hasil = x + y  # Penjumlahan
print(hasil)

Penjelasan: Di atas adalah contoh variabel dan operator yang digunakan dalam Python. Kode ini menjumlahkan x dan y, kemudian menampilkan hasilnya menggunakan fungsi print().
