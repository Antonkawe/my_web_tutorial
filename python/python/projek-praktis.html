Proyek Praktis: Kalkulator Sederhana

Berikut adalah contoh proyek Python untuk membuat kalkulator sederhana:


# Kalkulator Sederhana
def tambah(a, b):
    return a + b

def kurang(a, b):
    return a - b

def kali(a, b):
    return a * b

def bagi(a, b):
    return a / b

print("Tambah: ", tambah(5, 3))
print("Kurang: ", kurang(5, 3))
print("Kali: ", kali(5, 3))
print("Bagi: ", bagi(5, 3))

Penjelasan: Proyek ini menggunakan fungsi dasar untuk operasi matematika. Fungsi ini akan menerima dua argumen dan mengembalikan hasil operasi yang sesuai.
