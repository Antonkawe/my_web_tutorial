1. Metaprogramming dengan Decorators

Decorator adalah fungsi yang mengubah fungsi lain, memungkinkan pengguna untuk menambahkan fungsionalitas secara dinamis. Berikut contoh decorator untuk mengukur waktu eksekusi fungsi:


import time

def timer_decorator(func):
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        end = time.time()
        print(f"Time taken by {func.__name__}: {end - start:.4f} seconds")
        return result
    return wrapper

@timer_decorator
def calculate_prime_numbers(limit):
    primes = []
    for num in range(2, limit):
        if all(num % prime != 0 for prime in primes):
            primes.append(num)
    return primes

calculate_prime_numbers(5000)

Penjelasan: Decorator timer_decorator mengukur waktu yang dibutuhkan oleh fungsi calculate_prime_numbers untuk mengeksekusi.

2. Asynchronous Programming dengan Asyncio

Python asyncio digunakan untuk menangani operasi IO secara asinkron. Berikut contoh untuk melakukan HTTP requests secara paralel:


import asyncio
import aiohttp

async def fetch(url):
    async with aiohttp.ClientSession() as session:
        async with session.get(url) as response:
            return await response.text()

async def main():
    urls = ["https://example.com", "https://python.org", "https://openai.com"]
    tasks = [fetch(url) for url in urls]
    results = await asyncio.gather(*tasks)
    for result in results:
        print(result[:100])  # Print first 100 characters

asyncio.run(main())

Penjelasan: Kode ini melakukan request HTTP asinkron ke beberapa URL secara paralel menggunakan asyncio dan aiohttp.

3. Generator Function dengan Coroutines

Generator memungkinkan penghitungan nilai secara lazy. Berikut adalah implementasi generator untuk menghitung deret Fibonacci:


def fibonacci_generator():
    a, b = 0, 1
    while True:
        yield a
        a, b = b, a + b

fibo = fibonacci_generator()
for _ in range(10):
    print(next(fibo))

Penjelasan: Generator fibonacci_generator secara efisien menghitung elemen berikutnya dalam deret Fibonacci tanpa menyimpan seluruh nilai dalam memori.

4. Manipulasi Data dengan Pandas

Python pandas memudahkan manipulasi data, terutama dalam analisis data. Berikut contoh untuk mengelompokkan data dan menghitung statistik:


import pandas as pd

data = {
    'name': ['Alice', 'Bob', 'Charlie', 'David', 'Edward'],
    'age': [24, 27, 22, 32, 29],
    'score': [85, 92, 88, 79, 95]
}

df = pd.DataFrame(data)
grouped = df.groupby('age').mean()
print(grouped)

Penjelasan: Kode ini menggunakan pandas untuk membuat DataFrame dan menghitung nilai rata-rata berdasarkan umur.

5. Object-Oriented Programming dengan Metaclasses

Metaclass memungkinkan pengguna untuk memodifikasi pembuatan kelas. Berikut adalah contoh metaclass yang mencatat pembuatan kelas baru:


class LoggingMeta(type):
    def __new__(cls, name, bases, class_dict):
        print(f"Creating class {name}")
        return super().__new__(cls, name, bases, class_dict)

class Person(metaclass=LoggingMeta):
    def __init__(self, name):
        self.name = name

p = Person("Alice")

Penjelasan: Metaclass LoggingMeta akan mencetak pesan setiap kali kelas baru dibuat, memungkinkan logging selama inisialisasi kelas.
