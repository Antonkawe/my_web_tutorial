Map, Filter, dan Reduce di Python

Fungsi map(), filter(), dan reduce() digunakan untuk memanipulasi koleksi data dengan cara yang lebih fungsional. Berikut adalah contoh penggunaannya:


from functools import reduce

# map: Mengalikan setiap elemen dengan 2
numbers = [1, 2, 3, 4]
result_map = list(map(lambda x: x * 2, numbers))

# filter: Memilih angka genap
result_filter = list(filter(lambda x: x % 2 == 0, numbers))

# reduce: Menjumlahkan semua elemen
result_reduce = reduce(lambda x, y: x + y, numbers)

print(result_map)
print(result_filter)
print(result_reduce)

Penjelasan: map() digunakan untuk menerapkan fungsi ke setiap elemen, filter() untuk memfilter elemen yang memenuhi kondisi, dan reduce() untuk mengurangi elemen menjadi satu hasil.
