Rekursi di Python

Rekursi adalah metode pemrograman di mana fungsi memanggil dirinya sendiri. Berikut adalah contoh rekursi untuk menghitung faktorial:


# Fungsi rekursif untuk menghitung faktorial
def faktorial(n):
    if n == 1:
        return 1
    else:
        return n * faktorial(n - 1)

print(faktorial(5))  # Output: 120

Penjelasan: Fungsi faktorial memanggil dirinya sendiri hingga mencapai nilai dasar (n == 1), yang kemudian menghitung hasil faktorial.
